# Start API Server

Start the LiftLogic API server for development.

## Usage
```
/project:serve [--reload]
```

## Instructions

Start the FastAPI server:

```bash
cd /Users/fender/Desktop/liftlogic && liftlogic serve --reload
```

The API will be available at:
- API: http://localhost:8000
- Docs: http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc

Run this in the background so you can continue working.
